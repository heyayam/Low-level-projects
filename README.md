# Low Level Projects 

This repository contains a collection of projects focused on **low-level programming** and **system-level concepts**.

## 📌 About
The goal of this repository is to strengthen my understanding of how systems work under the hood by building projects using low-level languages like:
- C
- C++
- (later) Assembly

## Topics Covered
- Memory management
- Pointers and data structures
- File handling
- Process and system calls
- Basic operating system concepts
- Performance optimization

## 📂 Project Structure
Each project will have its own folder with:
- Source code
- Build/run instructions
- Explanation of concepts used
